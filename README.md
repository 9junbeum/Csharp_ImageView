# Csharp_ImageView

# 소개
C#을 이용하여 이미지뷰어를 작성
폴더를 선택하면, 폴더에 있는 이미지파일을 가지고 온다.
리스트에서 더블클릭하면 오른쪽에 사진이 출력됨.
